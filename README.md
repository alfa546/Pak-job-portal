## 📖 About the Project

**Pak Job Portal** is a full-featured, responsive job listing platform built specifically for the Pakistani job market. It allows job seekers to browse, search, and apply for jobs with ease — all in one place.

Whether you're a fresh graduate or an experienced professional, Pak Job Portal makes your job hunt simple, fast, and efficient.

---

## ✨ Features

### 👤 Job Seeker
- 🔍 **Search & Filter Jobs** — Search by title, location, category, or salary range
- 📄 **View Job Details** — Detailed job descriptions with requirements and benefits
- 📝 **Apply for Jobs** — Submit applications directly through the portal
- 🔖 **Save Jobs** — Bookmark interesting job listings for later
- 👤 **Profile Management** — Create and update personal profile and resume
- 📬 **Application Tracking** — Track the status of submitted applications
- 🔔 **Job Alerts** — Get notified about relevant new job postings

## 🚀 Getting Started

Follow these steps to run the project locally on your machine.

### ✅ Prerequisites

Make sure you have the following installed:

- [Node.js](https://nodejs.org/) (v16 or higher)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)
- [Git](https://git-scm.com/)

### 📥 Installation

1. **Clone the repository**

```bash
git clone https://github.com/alfa546/Pak-job-portal.git
```

2. **Navigate to the project directory**

```bash
cd Pak-job-portal
```

3. **Install dependencies**

```bash
npm install
# or
yarn install
```

4. **Set up environment variables**

Create a `.env.local` file in the root directory:

```env
NEXT_PUBLIC_API_URL=your_api_url_here
NEXT_PUBLIC_SITE_URL=http://localhost:3000
```

5. **Run the development server**

```bash
npm run dev
# or
yarn dev
```

6. **Open in browser**

```
http://localhost:3000
```

